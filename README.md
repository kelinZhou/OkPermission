# OkPermission
#### 实现Android6.0动态权限的优雅获取

* * *

## 简介
本库是使用Kotlin语音开发的。可以优雅的申请Android6.0中的动态权限。

## 下载
###### 第一步：添加 JitPack 仓库到你项目根目录的 gradle 文件中。
```
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```
###### 第二步：添加这个依赖。
```
dependencies {
    implementation 'com.github.kelinZhou:Okpermission:1.0.0'
}
```
